# Data Mining Project: Billionaires Analysis

## Project Overview
This project analyzes the world's top billionaires to uncover patterns, trends, and insights in wealth distribution. The goal is to understand demographics, net worth, industries, and geographical distribution.

## Key Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA) with visualizations
- Feature engineering and insights generation
- Summary of top billionaires by net worth, industry, and country

## Visualizations
- Distribution of billionaire net worth
- Top countries with most billionaires
- Industry-wise wealth comparison
- Trend analysis over years

## Results
- The United States has the highest number of billionaires.
- Technology and finance sectors dominate wealth accumulation.
- Net worth is heavily concentrated among a few top individuals.

## Project Files
- `Billionaires_Analysis.ipynb` : The main notebook with all analysis
- `data/billionaires.csv` : Dataset used for analysis
- `images/` : Folder containing all visualizations
